Este proyecto consiste en el desarrollo de una página web. El objetivo principal es aplicar los conocimientos básicos de HTML, CSS y estructura web adquiridos durante el curso para crear una página funcional, organizada y visualmente coherente.

El sitio web está compuesto por varias secciones que integran contenido personal, académico y temático. Incluye una página principal, un formulario de contacto, una sección temática, una página dedicada al primer año del grado, una página con información detallada sobre la asignatura de fundamentos de la ingeniería informática basada en la guía docente, una sección “Acerca de mí” con currículum e imágenes, y una red de enlaces a las páginas web de compañeros de clase.

Han surgido muchos problemas a lo largo del trabajo. Algunos de ellos han sido los href (Para poder conectar las páginas) y conseguir crear el encabezado.

Este proyecto demuestra la capacidad para aplicar los fundamentos del desarrollo web, organizar información de forma estructurada y presentar contenido académico de manera clara y profesional, integrando tanto habilidades técnicas como competencias transversales adquiridas durante la asignatura.
